Installation
============

::

  pip install "git+https://github.com/demosdemon/teragested.git#egg=teragested"

Documentation
=============

To build the docs, run::

  make docs

or visit, https://teragested.readthedocs.io/.

Development
===========

To run all the tests, run::

  make test
